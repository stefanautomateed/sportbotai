## ðŸ“‹ Quick Reference: Brand Guidelines

### Colors (Use in EVERY prompt):
```
Primary Background: #0A0A0A (pure black)
Accent Color: #0D9488 (teal/cyan) - ONLY accent color
Text Primary: #FFFFFF (white)
Text Secondary: #9CA3AF (gray)
Card Borders: rgba(255,255,255,0.05) (subtle white)
Glass Effect: backdrop-blur with 5% white border
```

### Visual Style Checklist:
- [ ] Black background in all app screens
- [ ] Teal accent used for all interactive elements
- [ ] Glass-morphism cards (not flat)
- [ ] Modern sans-serif typography
- [ ] Generous spacing and padding
- [ ] Smooth, professional animations
- [ ] NO other accent colors (no blue, no bright green, no red except for warnings)

### Animation Style:
- Smooth and professional (not jumpy)
- Apple-style transitions (fade, scale, slide)
- Fast enough to maintain energy
- Slow enough to be elegant
- Always purposeful, never decorative

### Reference Brands for Style:
- **Apple**: Premium feel, smooth animations, minimalist
- **Notion**: Clean UI, glass effects, modern
- **Stripe**: Dark theme, teal accents, professional
- **Linear**: Fast, smooth, confidence-inspiring

## ðŸ’¡ Tips for Better Results

### When Writing to InVideo:
1. **Copy entire prompt** - don't summarize
2. **Paste as-is** - InVideo AI reads long prompts well
3. **Generate first** - see what it creates
4. **Then iterate** - ask for adjustments if needed

### Common Adjustments:
- "Make the background darker, pure black"
- "Change accent color from blue to teal cyan #0D9488"
- "Add more spacing between elements"
- "Make animations smoother and slower"
- "Show phone screen more clearly"

### For Best Output Quality:
- Export at 1080p minimum (4K if available)
- Use H.264 codec for compatibility
- 30fps for smooth playback
- High bitrate for quality
